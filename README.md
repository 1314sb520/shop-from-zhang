#### 🚧 安装 yarn

- 复制代码（终端运行） `npm install -g yarn`
- 建议使用yarn，npm可能会出现请求比较慢或者丢包的情况

```bash
# 安装依赖
npm install

# 运行项目
npm run dev

# 打包发布
npm run build
```
